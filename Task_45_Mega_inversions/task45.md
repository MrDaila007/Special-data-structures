# Задача 45. Мегаинверсии

- **Имя входного файла:** input.txt
- **Имя выходного файла:** output.txt
- **Ограничение по времени:** 1 с
- **Ограничение по памяти:** 256 МБ

Мегаинверсией перестановки $p_1, p_2, \ldots, p_n$ назовём тройку индексов $(i, j, k)$, для которой $i < j < k$ и $p_i > p_j > p_k$. Требуется посчитать количество мегаинверсий.

## Формат входных данных

В первой строке задано целое число $n$ ($1 \le n \le 300\,000$).

В следующих $n$ строках записана перестановка: в $i$-й строке находится число $p_i$ ($1 \le p_i \le n$). Все элементы различны.

## Формат выходных данных

Выведите одно число — количество мегаинверсий в перестановке.
