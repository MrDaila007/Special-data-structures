# Задача 1.2. Минимум в списке

- **Имя входного файла:** стандартный ввод
- **Имя выходного файла:** стандартный вывод
- **Ограничение по времени:** 1 с
- **Ограничение по памяти:** 128 МБ

Дан пустой список $L$ и операции над ним:

1. `+L x` — вставить число $x$ в начало списка ($1 \le x \le 10^9$);
2. `+R x` — вставить число $x$ в конец списка ($1 \le x \le 10^9$);
3. `-L` — удалить элемент из начала списка;
4. `-R` — удалить элемент из конца списка.

После каждой операции нужно вывести текущий минимальный элемент списка.

## Формат входных данных

Первая строка содержит целое число $q$ ($1 \le q \le 10^6$) — количество операций.

В следующих $q$ строках записаны операции в описанном формате. Гарантируется, что перед любой операцией удаления список непуст.

## Формат выходных данных

Выведите $q$ чисел — минимальный элемент списка после каждой операции. Если список пуст, выведите `-1`.

## Пример

**Ввод**

```text
4
+L 5
+R 3
-L
+R 1
```

**Вывод**

```text
5
3
3
1
```
