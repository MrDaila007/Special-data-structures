# Задача Картина

- **Имя входного файла:** стандартный ввод
- **Имя выходного файла:** стандартный вывод
- **Ограничение по времени:** 1,5 с
- **Ограничение по памяти:** 256 МБ

Художник создаёт одномерные чёрно-белые картины и позволяет посетителям задавать параметры полос. Каждая полоса описывается цветом ($W$ — белый, $B$ — чёрный), начальной позицией и длиной. Полотно изначально белое. После обработки каждого пожелания нужно знать количество чёрных полос и их суммарную длину.

## Формат входных данных

Первая строка содержит количество запросов $n$ ($1 \le n \le 10^5$). В каждой из следующих $n$ строк записаны параметры полосы: $c_i$, $l_i$, $x_i$ — цвет, начало и длина. Значения $l_i$ и $x_i$ целые, причём $x_i + l_i \le 5 \cdot 10^5$, $l_i \ge 0$, $x_i \ge 1$.

## Формат выходных данных

После каждого запроса выведите в отдельной строке количество чёрных полос и их суммарную длину, разделённые пробелом.

## Пример

**Ввод**

```text
7
W 2 3
B 2 2
B 4 2
B 3 2
B 7 2
W 3 1
W 0 10
```

**Вывод**

```text
0 0
1 2
1 4
1 4
2 6
3 5
0 0
```
