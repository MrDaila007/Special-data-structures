# Задача Перестановки

- **Имя входного файла:** стандартный ввод
- **Имя выходного файла:** стандартный вывод
- **Ограничение по времени:** 2 с
- **Ограничение по памяти:** 256 МБ

Последовательность длины $n$ называется перестановкой, если она содержит каждое число от $1$ до $n$ ровно один раз. Загадана перестановка $p$. Для каждого индекса $i$ известно число $c_i$ — количество таких $j < i$, что $p_j > p_i$ (то есть количество инверсий с участием элемента на позиции $i$). Восстановите перестановку.

## Формат входных данных

Первая строка содержит целое число $n$ ($1 \le n \le 2 \cdot 10^5$) — размер перестановки.

Вторая строка содержит $n$ целых чисел $c_1, c_2, \ldots, c_n$ ($0 \le c_i < n$).

## Формат выходных данных

Выведите $n$ целых чисел — восстановленную перестановку.

## Полезный комментарий

Компилятор: Microsoft Visual C++ 15.9 (2017) x64.

Чтобы ускорить ввод-вывод, используйте:

```cpp
int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
}
```

## Пример

**Ввод**

```text
3
0 0 0
```

**Вывод**

```text
1 2 3
```
