# Задача 3.2. Проверка ответа

- **Имя входного файла:** стандартный ввод
- **Имя выходного файла:** стандартный вывод
- **Ограничение по времени:** 1 с
- **Ограничение по памяти:** 256 МБ

В предыдущей задаче требовалось вывести эйлеров обход дерева. Проверьте, может ли заданная последовательность вершин быть корректным эйлеровым обходом некоторого дерева.

## Формат входных данных

В единственной строке записано нечётное число $k$ ($3 \le k = 2 \cdot n - 1 \le 199\,999$) целых чисел $a_i$ ($1 \le a_i \le n$) — последовательность вершин. Числа разделены одиночными пробелами, пробелы в начале и в конце строки отсутствуют.

## Формат выходных данных

Выведите `YES`, если последовательность является эйлеровым обходом некоторого дерева, иначе выведите `NO`.

Компилятор: Microsoft Visual C++ 15.9 (2017) x64.
