# Задача Лотерея

- **Имя входного файла:** input.txt
- **Имя выходного файла:** output.txt
- **Ограничение по времени:** 2 с
- **Ограничение по памяти:** 64 МБ

В городе $\chi$ проводится лотерея, напоминающая «Спортлото 5 из 36». Каждый билет состоит из двух частей и содержит $N$ чисел. Игрок вычёркивает как минимум одно число, отправляет заполненную часть организаторам, а вторую оставляет себе. Организаторы определяют два числа $L$ и $R$ ($L \le R$); билет считается выигрышным, если среднее арифметическое вычеркнутых чисел попадает в отрезок $[L, R]$.

Известно, что тираж уже состоялся, числа $L$ и $R$ известны, и у вас остался незаполненный билет. Требуется посчитать, сколькими способами можно было вычеркнуть числа, чтобы билет оказался выигрышным.

## Формат входных данных

Первая строка содержит целое число $N$ ($1 \le N \le 36$) — количество чисел в билете.

Во второй строке записаны целые числа $A_i$ ($-10^{15} \le A_i \le 10^{15}$), разделённые пробелами.

В третьей строке находятся целые числа $L$ и $R$ ($-10^{15} \le L \le R \le 10^{15}$), разделённые пробелом.

## Формат выходных данных

Выведите одно число — количество способов выбрать набор чисел, чтобы их среднее арифметическое принадлежало диапазону $[L, R]$. Два способа различаются, если существует индекс $i$, для которого число $A_i$ вычеркнуто в одном случае и оставлено в другом.

## Примеры

### Пример 1

**Ввод (`input.txt`)**

```text

0 1

```

**Вывод (`output.txt`)**

```text
1
```

### Пример 2

**Ввод (`input.txt`)**

```text
2
0 1

```

3
