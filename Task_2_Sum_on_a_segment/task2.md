# Задача 2. Сумма на отрезке

- **Имя входного файла:** стандартный ввод
- **Имя выходного файла:** стандартный вывод
- **Ограничение по времени:** 2 с
- **Ограничение по памяти:** 512 МБ

Дан массив $A$ из $n$ положительных чисел $a_i$ ($1 \le a_i \le 10^9$). Необходимо обрабатывать запросы четырёх видов:

1. `1 p v` — присвоить $p$-му элементу значение $v$ ($1 \le v \le 10^9$);
2. `2 l r` — увеличить на $1$ все элементы на отрезке $[l, r]$;
3. `3 l r` — найти сумму чётных чисел на отрезке $[l, r]$;
4. `4 l r` — найти сумму нечётных чисел на отрезке $[l, r]$.

## Формат входных данных

Первая строка содержит числа $n$ и $q$ ($1 \le n, q \le 10^5$) — размер массива и количество запросов.

Вторая строка содержит $n$ целых чисел $a_i$.

Каждая из следующих $q$ строк описывает один запрос в указанном формате.

## Формат выходных данных

Для каждого запроса типов `3` и `4` выведите ответ — значение суммы на соответствующем отрезке.

## Полезный комментарий

Компилятор: Microsoft Visual C++ 15.9 (2017) x64.

Чтобы ускорить ввод-вывод, используйте:

## Пример 1

**Ввод**

```text

1 2 3 4
3 1 4
2 1 3
4 1 4
3 1 4
1 4 10
3 1 3
3 2 4
4 1 3
4 2 4
```

**Вывод**

```text
6
3
10
6
14
3
3
```

## Пример 2

**Ввод**

```text

```

**Вывод**

```text

9
```
