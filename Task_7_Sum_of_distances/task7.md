# Задача 7. Сумма расстояний

- **Имя входного файла:** стандартный ввод
- **Имя выходного файла:** стандартный вывод
- **Ограничение по времени:** 2 с
- **Ограничение по памяти:** 256 МБ

Дан ориентированный граф $G$. Обозначим длину кратчайшего пути из вершины $u$ в вершину $v$ через $d(u, v)$. Если пути из $u$ в $v$ не существует, положим $d(u, v) = n$. Требуется найти сумму квадратов длин кратчайших путей по всем парам вершин графа:
$$
\sum_{1 \le u, v \le n} d^2(u, v).
$$

Совет: используйте алгоритм, основанный на битовых представлениях множеств (хотя другие корректные подходы тоже подходят).

## Формат входных данных

Первая строка содержит число $n$ ($1 \le n \le 2000$) — количество вершин графа $G$.

Далее следует матрица смежности $E$ размера $n \times n$. В $i$-й строке находится $n$ символов `0` или `1`: элемент равен `1`, если дуга из $i$ в $j$ существует, и `0` иначе. Петли допускаются, поэтому $E_{ii}$ может быть равен `1`.

## Формат выходных данных

Выведите одно целое число — сумму квадратов кратчайших расстояний по всем упорядоченным парам вершин графа $G$.

## Полезный комментарий

Компилятор: Microsoft Visual C++ 15.9 (2017) x64.

Чтобы ускорить ввод-вывод, добавьте:

```cpp
int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
}
```

**Ввод**

```text

001
100
```

**Вывод**

```text
15
```
