# Задача Билеты

- **Имя входного файла:** стандартный ввод
- **Имя выходного файла:** стандартный вывод
- **Ограничение по времени:** 1 с
- **Ограничение по памяти:** 64 МБ

В новых электричках у каждого пассажира должно быть сидячее место. Количество мест ограничено, поэтому билет продаётся только если на всём выбранном участке пути остаётся хотя бы одно свободное место. Электричка проходит через $N$ станций, пронумерованных от $0$ до $N-1$. Покупатель билета задаёт два числа $x$ и $y$ — номера станций отправления и назначения.

Необходимо обработать последовательность запросов в порядке поступления.

## Формат входных данных

В первой строке записаны три натуральных числа $N$, $K$ и $M$ — количество станций, количество сидячих мест и число запросов соответственно ($2 \le N \le 2 \cdot 10^6$, $1 \le K \le 10^4$, $1 \le M \le 10^5$).

В следующих $M$ строках описаны запросы, каждый состоит из двух целых чисел $x$ и $y$ ($0 \le x < y < N$).

## Формат выходных данных

Для каждого запроса выведите в отдельной строке `1`, если билет был продан, и `0` в противном случае.
