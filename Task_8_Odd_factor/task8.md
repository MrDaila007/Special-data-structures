# Задача 8. Нечётный фактор

- **Имя входного файла:** oddfactor.in
- **Имя выходного файла:** oddfactor.out
- **Ограничение по времени:** 1 с
- **Ограничение по памяти:** 256 МБ

Нечётным фактором неориентированного графа называется остовный подграф (то есть подграф, содержащий все вершины исходного графа), в котором степень каждой вершины нечётна.

Дан неориентированный граф. Постройте любой его нечётный фактор или определите, что он не существует. Граф может состоять из нескольких компонент связности.

## Формат входных данных

В первой строке заданы числа $n$ и $m$ — количество вершин ($1 \le n \le 10^5$) и рёбер ($0 \le m \le 2 \cdot 10^5$). В каждой из следующих $m$ строк записано ребро — два числа $a_i$ и $b_i$ ($1 \le a_i, b_i \le n$, $a_i \ne b_i$), обозначающие вершины, которые оно соединяет.

## Формат выходных данных

Если нечётный фактор существует, в первой строке выведите количество рёбер в нём, а затем перечислите сами рёбра (по одному в строке). Если построить нечётный фактор невозможно, выведите `-1`.
