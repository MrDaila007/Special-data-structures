# Задача Лунки

- **Имя входного файла:** стандартный ввод
- **Имя выходного файла:** стандартный вывод
- **Ограничение по времени:** 0,5 с
- **Ограничение по памяти:** 256 МБ

Петя играет в «Лунки». В ряд расположено $N$ лунок, у каждой есть сила выброса $a_i$. Если бросить шарик в лунку $i$, он сразу вылетает и попадает в лунку $i + a_i$. Процесс продолжается, пока шарик не вылетит за край ряда. На каждом из $M$ ходов Петя либо меняет силу выброса одной лунки, либо бросает шарик и хочет знать количество прыжков и номер последней лунки перед вылетом.

## Формат входных данных

Первая строка содержит числа $N$ и $M$ ($1 \le N \le 10^5$, $1 \le M \le 10^5$) — количество лунок и число ходов.

Вторая строка содержит $N$ положительных чисел, не превышающих $N$ — изначальные силы выброса $a_i$.

Далее следуют $M$ строк с описанием ходов:

- `0 a b` — установить силу выброса лунки $a$ равной $b`;
- `1 a` — бросить шарик в лунку $a$.

Числа $a$ и $b$ целые, положительные и не превосходят $N$.

## Формат выходных данных

Для каждого хода второго типа выведите в отдельной строке два числа — номер последней лунки перед вылетом шарика и количество прыжков.

## Пример

**Ввод**

```text
8 5
1 1 1 1 1 2 8 2
1 1
0 1 3
1 1
0 3 4
1 2
```

**Вывод**

```text
8 7
8 5
7 3
```
