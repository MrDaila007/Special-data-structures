# Задача 10. Очередь в банк

- **Имя входного файла:** стандартный ввод
- **Имя выходного файла:** стандартный вывод
- **Ограничение по времени:** 3 с
- **Ограничение по памяти:** 256 МБ

В банке действует электронная очередь. Каждый клиент выбирает время прихода и длительность визита. Клиенты обслуживаются по времени записи, но каждый ждёт завершения всех визитов перед ним. Клиент может отменить свой визит.

Сотруднику Ивану нужно узнать, сколько времени ему придётся ждать, если он придёт в указанные моменты. Помогите ему вычислить ответ на каждый запрос.

## Формат входных данных

Первая строка содержит целое число $q$ ($1 \le q \le 3 \cdot 10^5$) — количество событий. Далее следуют события трёх типов:

- `+ t d` ($1 \le t, d \le 10^6$) — новый клиент приходит во время $t$ и обслуживается $d$ единиц времени;
- `- i` ($1 \le i \le q$) — отмена визита клиента из $i$-го события типа `+`, который ещё не был отменён;
- `? t` ($1 \le t \le 10^6$) — Иван интересуется временем ожидания, если придёт в момент $t$.

Гарантируется, что после каждого события в очереди не окажется двух клиентов с одинаковым временем прихода. События `-` всегда ссылаются на ранее записанные и ещё не отменённые события `+`. Если Иван приходит одновременно с клиентом, он пропускает клиента вперёд.

## Формат выходных данных

Для каждого события `?` выведите в отдельной строке время ожидания Ивана.
