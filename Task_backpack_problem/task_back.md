# Задача 1. Задача о рюкзаке

- **Имя входного файла:** стандартный ввод
- **Имя выходного файла:** стандартный вывод
- **Ограничение по времени:** 4 с
- **Ограничение по памяти:** 512 МБ

Даны $n$ предметов, каждый с весом $w_i$ и ценностью $p_i$. Нужно выбрать подмножество предметов суммарным весом не более $W$, чтобы суммарная ценность была максимальной.

## Формат входных данных

Первая строка содержит два числа $n$ и $W$ ($1 \le n \le 40$, $1 \le W \le 10^9$) — количество предметов и максимальный допустимый вес.

Далее следуют $n$ строк, каждая содержит по два числа $w_i$ и $p_i$ ($1 \le w_i, p_i \le 10^9$) — вес и ценность $i$-го предмета.

## Формат выходных данных

В первой строке выведите число $k$ ($0 \le k \le n$) — количество выбранных предметов.

Во второй строке перечислите $k$ различных целых чисел от $1$ до $n$ — номера выбранных предметов (в порядке их появления во входных данных). Если оптимальных решений несколько, разрешается вывести любое.

## Полезный комментарий

Компилятор: Microsoft Visual C++ 15.9 (2017) x64.

Чтобы ускорить ввод-вывод, добавьте:

```cpp
int main() {

```

**Ввод**

```text

```

**Вывод**

```text
3

```
