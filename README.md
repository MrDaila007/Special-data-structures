# Special Data Structures

Репозиторий с реализациями лабораторных работ по курсу «Специальные структуры данных». Для каждой задачи сохранены исходники решений на C++17, описания условий и вспомогательные материалы.

## Как устроен репозиторий
- Каждый каталог `Task_*` или с номером задачи содержит три основных файла: `task*.md` (условие), `task*.cpp` (решение) и `run_tests.sh` (локальный запуск проверки).
- Скрипты сборки используют `g++ -std=c++17 -O2`; дополнительные тесты размещаются в подкаталоге `tests` рядом со скриптом (если предоставлены).
- Некоторые задачи включают иллюстрации (`*.png`) или подготовленные бинарные файлы проверяющих программ.
- В корне лежит файл `LICENSE` с условиями распространения решений.

## Запуск решения и тестов
1. Перейдите в каталог нужной задачи, например `Task_1_1_Add_one_many_times/`.
2. Убедитесь, что скрипт исполняемый: `chmod +x run_tests.sh` (нужно один раз).
3. Запустите `./run_tests.sh`. Скрипт соберёт решение, прогонит его на всех тестах из подпапки `tests` (если она есть) и сообщит о результате.

### Запуск всех задач разом
В корне проекта доступен скрипт `./run_all_tests.sh`. Он последовательно пройдёт по всем каталогам, где есть `run_tests.sh`, выполнит проверки и выведет сводку по успешным и упавшим задачам. Код возврата будет ненулевым, если хотя бы одна задача завершилась с ошибкой.
