Имеется двоичный счетчик из $k$ бит в начальном состоянии $b_{1}b_{2} \ldots b_{k}$.Выполним операцию инкремента $n$ раз.Определите наибольшее число изменившихся бит за одну операции инкремента и среднее число изменившихся битов.Формат входных данныхВ первой строке записано целое число $k$ ($1 \le k \le 60$).Во второй строке записано начальное состояние счетчика $b_{1}$, $b_{2}$ …$b_{k}$ — 0 или 1.В третьей строке записано число $n$ ($1 \le n \le 2^{60}-1$).Известно, что двоичный счетчик не будет переполняться, т.е. выполняться инкремент с состоянию из всех 1.Формат выходных данныхВ первой строке выведите наибольшее число изменившихся бит за одну операции инкремента.Во второй строке выведите среднее число изменившихся битов за $n$ операций в виде несократимой дроби A/B.Примеры

| 5                  | 5                  |
