# Задача 1.1. Добавить один много раз

- **Имя входного файла:** стандартный ввод
- **Имя выходного файла:** стандартный вывод
- **Ограничение по времени:** 1 с
- **Ограничение по памяти:** 256 МБ

Дан двоичный счётчик длины $k$ бит в начальном состоянии $b_1 b_2 \ldots b_k$. Выполните операцию инкремента $n$ раз. Требуется определить:

- максимальное количество битов, изменившихся за одну операцию инкремента;
- среднее количество изменённых битов за все $n$ операций (в виде несократимой дроби).

Гарантируется, что переполнения счётчика (перехода из состояния всех единиц) не произойдёт.

## Формат входных данных

Первая строка содержит целое число $k$ ($1 \le k \le 60$).

Во второй строке записано начальное состояние счётчика — последовательность из $k$ чисел $b_i$ (каждое равно `0` или `1`).

В третьей строке находится число $n$ ($1 \le n \le 2^{60} - 1$).

## Формат выходных данных

В первой строке выведите максимальное число битов, которые менялись в одной операции инкремента.

Во второй строке выведите среднее число изменённых битов за $n$ операций в виде несократимой дроби `A/B`.

## Примеры

### Пример 1

**Ввод**

```text
3
0 0 0
5
```

**Вывод**

```text
3
8/5
```

### Пример 2

**Ввод**

```text
5
0 1 1 1 1
1
```

**Вывод**

```text
5
5/1
```

### Пример 3

**Ввод**

```text
5
0 1 1 1 1
3
```

**Вывод**

```text
5
8/3
```
