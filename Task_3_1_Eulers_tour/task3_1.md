# Задача 3.1. Эйлеров обход

- **Имя входного файла:** стандартный ввод
- **Имя выходного файла:** стандартный вывод
- **Ограничение по времени:** 1 с
- **Ограничение по памяти:** 256 МБ

Дано дерево $T$. Требуется вывести любой его эйлеров обход — последовательность вершин, полученную при обходе в глубину, где вершина записывается при каждом посещении (при входе и при возвращении по ребру).

![Task_3_1_Eulers_tour.png](./Task_3_1_Eulers_tour.png)

## Формат входных данных

Первая строка содержит число $n$ ($2 \le n \le 100\,000$) — количество вершин дерева.

Каждая из следующих $n - 1$ строк содержит два числа $a_i$ и $b_i$ ($1 \le a_i, b_i \le n$, $a_i \ne b_i$) — описание ребра.

Гарантируется, что заданный граф является деревом.

## Формат выходных данных

Выведите $2 \cdot n - 1$ чисел — эйлеров обход дерева.
