# Задача Xor-пути

- **Имя входного файла:** стандартный ввод
- **Имя выходного файла:** стандартный вывод
- **Ограничение по времени:** 3 с
- **Ограничение по памяти:** 256 МБ

Задано прямоугольное поле размера $n \times m$. В каждой клетке $(i, j)$ написано целое число $a_{i,j}$. Необходимо посчитать количество путей из $(1,1)$ в клетку $(n,m)$, которые удовлетворяют условиям:

- из клетки можно перемещаться только вниз или вправо, то есть из клетки $(i,j)$ разрешены переходы в $(i+1,j)$ или $(i,j+1)$, при этом нельзя выходить за пределы поля;
- xor всех чисел на выбранном пути равен ровно $k$.

Найдите количество подходящих путей.

## Формат входных данных

Первая строка содержит три целых числа $n$, $m$ и $k$ — высоту и ширину поля и требуемое значение xor ($1 \le n, m \le 20$, $0 \le k \le 10^{18}$).

Следующие $n$ строк содержат по $m$ целых чисел. В $i$-й строке $j$-й элемент равен $a_{i,j}$ ($0 \le a_{i,j} \le 10^{18}$).

## Формат выходных данных

Выведите одно целое число — количество путей из $(1,1)$ в $(n,m)$, для которых xor всех чисел на пути равен $k$.
