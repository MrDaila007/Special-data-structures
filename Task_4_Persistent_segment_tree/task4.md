# Задача 4. Персистентное дерево отрезков

- **Имя входного файла:** стандартный ввод
- **Имя выходного файла:** стандартный вывод
- **Ограничение по времени:** 3 с
- **Ограничение по памяти:** 256 МБ

Дан массив $A$ из $n$ элементов. Необходимо обработать $q$ запросов. Для каждого запроса заданы числа $l_i$, $r_i$, $x_i$, $y_i$. Нужно определить, сколько элементов массива на отрезке $[l_i, r_i]$ находятся в диапазоне $[x_i, y_i]$.

## Формат входных данных

Первая строка содержит два числа $n$ и $q$ ($1 \le n, q \le 2 \cdot 10^5$) — размер массива и количество запросов.

Вторая строка содержит $n$ чисел $a_i$ ($1 \le a_i \le 10^9$) — элементы массива.

Далее следуют $q$ строк, каждая из которых содержит четыре числа $l_i$, $r_i$, $x_i$, $y_i$ ($1 \le l_i \le r_i \le n$, $1 \le x_i \le y_i \le 10^9$).

## Формат выходных данных

Выведите $q$ строк — ответы на запросы.
